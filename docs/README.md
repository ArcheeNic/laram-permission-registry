# Permission Registry Documentation

Эта директория содержит документацию для модуля Permission Registry.

## Файлы

### API.md
Подробная документация по REST API с примерами использования, описанием endpoints, параметров и ответов.

**Рекомендуется для:**
- Быстрого ознакомления с API
- Копирования примеров cURL команд
- Понимания концепций auto_grant/auto_revoke

### permission-registry-api.json
OpenAPI 3.0 спецификация в формате JSON для импорта в инструменты тестирования API.

**Поддерживаемые инструменты:**
- Apidog
- Postman
- Insomnia
- Swagger UI
- Redoc

**Как использовать:**
1. Откройте ваш API клиент (Postman, Apidog и т.д.)
2. Выберите "Import" / "Импорт"
3. Выберите файл `permission-registry-api.json`
4. Все endpoints, схемы и примеры будут автоматически импортированы

## Обновление документации

При изменении API обязательно обновите оба файла:
1. `permission-registry-api.json` - для автоматических инструментов
2. `API.md` - для human-readable документации
